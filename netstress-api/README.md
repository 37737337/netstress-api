\# NetStress API 🚀



REST API микросервис для автоматизированного нагрузочного тестирования веб-серверов. Проект объединяет удобство Python/FastAPI с мощностью системных Linux-утилит (`wrk`) для анализа пропускной способности и отказоустойчивости.



\## 🛠 Стек технологий

\* \*\*Backend:\*\* Python 3.11, FastAPI, Uvicorn

\* \*\*Infrastructure:\*\* Docker, Docker Compose, Linux Subprocess

\* \*\*Testing Tool:\*\* `wrk` (HTTP benchmarking tool)



\## ⚡ Особенности

\- Интеграция системных вызовов Linux напрямую из Python-кода.

\- Асинхронная обработка запросов.

\- Изолированная среда выполнения (Python + Linux-утилиты собраны в один Docker-контейнер).

\- Автоматический парсинг текстового вывода консольных утилит с помощью регулярных выражений.



\## 🚀 Как запустить (через Docker)



Вам не нужно устанавливать `wrk` или настраивать виртуальное окружение. Всё упаковано в контейнер.
